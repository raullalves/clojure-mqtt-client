# clojure mqtt client

Clojure MQTT client for hackaton

## Installation

Install Leiningen: https://leiningen.org/

    $ lein run

## Usage

Generate a JAR, put on a OSGi container, or run it

    $ lein uberjar
    $ java -jar irrigation-client-0.1.0-standalone.jar
